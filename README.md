반자동화 가계부 어플리케이션
====
개발자: 김준하

주요기능
----
+ 현금 이외의 다양한 포인트를 자산으로 추가할 수 있음
+ 저장된 정보는 인터넷 공유 없이 각 기기의 로컬 데이터베이스에 안전하게 저장됨
+ 지속적인 수입이 지정된 경우 입력한 날짜의 주기별로 자동 수입기록 (매일, 매주, 매월, 매 분기, 매년)
+ 지출내역 테그로 효과적인 분석 가능
+ 각 수입 항목별 데이터 시각화 (파이차트, 바차트, 라인차트)
